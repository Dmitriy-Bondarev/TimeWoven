# temp/

Папка для временных файлов: дампы, экспорты, отладочные скрипты, SQL-черновики.

> **Важно:** содержимое этой папки **не версионируется** (см. `.gitignore`).
> Файлы здесь носят временный характер и не должны попадать в репозиторий.

## Типичное использование

| Тип файла | Пример |
|-----------|--------|
| SQL-дампы | `pg_dump_2026-04-17.sql` |
| CSV-экспорты | `people_export_2026-04-17.csv` |
| Отладочные скрипты | `debug_relations.py` |
| Черновики миграций | `migration_draft_v2.sql` |

## Очистка

Периодически очищай папку вручную или командой:
```bash
find temp/ -type f ! -name '.gitkeep' ! -name 'README.md' -delete
```
